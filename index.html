Hands-on Learning

Lab Sessions

Five progressive labs building from basic UI to full local database persistence using Jetpack Compose and the Android ecosystem.

Lab 01

UI Fundamentals

Building basic UI elements using Jetpack Compose, exploring modifiers and layout composables to structure simple screens.

Reflection I learned the fundamentals of Jetpack Compose by building basic UI elements. Understanding how modifiers and layouts work was key to structuring simple screens.
Lab 02

Interaction & State

Exploring state management and user interactions to create dynamic, input-responsive interfaces in Compose.

Reflection I explored state management and user interactions. This lab taught me how to make the UI dynamic and responsive to user inputs.
Lab 03

Material Design

Applying Material Design 3 guidelines and standard components to create polished, professional-looking Android UIs.

Reflection I applied Material Design guidelines to improve the app's aesthetics and user experience. Utilizing standard components helped create a polished and professional look.
Lab 04

Navigation

Implementing multi-screen navigation using Jetpack Compose Navigation, including a bottom navigation bar for improved UX flow.

Reflection I implemented multi-screen navigation using Jetpack Compose Navigation. Learning to set up a bottom navigation bar significantly improved the app's structure and flow.
Lab 05

Room Database

Integrating the Room database for local data persistence, with ViewModel and dependency injection for clean architecture.

Reflection I integrated the Room database for local data persistence. This was a crucial step in understanding how to save, retrieve, and manage data efficiently on the device using ViewModels and dependency injection.
